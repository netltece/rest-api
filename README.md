# Installation
```bash
git clone https://github.com/netltece/rest-api.git
```
```bash
docker-compose up -d
```
```bash
docker-compose exec php-fpm bash
```
```bash
composer install
```
```bash
cp .env.example .env
```
```bash
php artisan key:generate
```
```bash
php artisan migrate
```
```bash
php artisan db:seed
```
